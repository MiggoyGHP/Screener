Stock Pattern Screener

Algorithmic detection of swing trading setups — VCPs, Reversals, Resets, and Coils — with Relative Strength filtering and interactive Streamlit dashboard.

Python 3.11+ Streamlit Dashboard RS Line Integration Free Market Data

Detected Patterns

VCP

Volatility Contraction Pattern (Minervini / Qullamaggi). Detects tightening price contractions with volume dry-up. Requires full Trend Template pass.

Reversal

Stage 1 to Stage 2 transition. Identifies stocks breaking out of prolonged downtrends with trendline breaks, MA reclaims, and volume confirmation.

Reset

Pullback to moving average in an uptrend. Detects orderly pullbacks to the 10 EMA, 21 EMA, or 50 SMA with volume contraction and bounce.

Coil / Darvas Box

Tight price consolidation with ATR contraction and volume dry-up. Identifies stocks coiling in a narrow range, ready to break out.

Features

RS

Relative Strength Line

Stock vs SPY comparison with RS ranking, new-high detection, and chart overlay. Top 30% filter for VCPs.

TT

Minervini Trend Template

8-criteria Stage 2 uptrend filter. Price vs MAs, MA alignment, 52-week position, and RS rank.

SC

Composite Scoring

Weighted scoring: pattern quality (50%), RS rank (20%), volume (15%), pivot proximity (15%). Ranked results.

CH

Interactive Charts

Candlestick charts with MA overlays, volume, RS line panel, and pattern annotations. Visual confirmation built in.

Quick Start

git clone https://github.com/MiggoyGHP/Screener.git
cd Screener
pip install -e .
cd dashboard
streamlit run app.py

Three dashboard pages: Screener Results (full market scan), Pattern Explorer (single ticker deep-dive), Parameter Tuner (adjust detection thresholds).
